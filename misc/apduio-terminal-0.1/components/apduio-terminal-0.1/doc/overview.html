Apduio CardTerminal API specification.

This document is the API specification of the Gemplus CardTerminal package for accessing the JavaCard simulator distributed by Sun in its JavaCard Development Kit.

See http://java.sun.com/products/javacard/dev_kit.html for more details on the JavaCard Development Kit.

Apduio CardTerminal (com.gemplus.opencard.terminal.apduio), version 0.1.
