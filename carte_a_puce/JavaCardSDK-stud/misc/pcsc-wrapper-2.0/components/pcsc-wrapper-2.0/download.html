PC/SC CardTerminal Download Page

This page describes how to download, and install the PC/SC CardTerminal component that allows accessing Smart Cards through a PC/SC resource manager (a PC/SC driver must have been installed for your card reader!).

See also the Generic CardServices Home-Page for a complete on-line documentation set about this component.

This is a first version and is provided AS-IS. See the Important Note section below.

Distribution

or

Two distribution files are available: one in "tar.gz" format, and one in "zip" format. Select the one corresponding to your platform and/or available unpacking tools, e.g.,

  • if you are under Unix, download: "gemplus-service-0.2.tar.gz"

  • if you are under Windows, download: "gemplus-service-0.2.zip"

Note that the "tar.gz" file is significantly smaller (30% smaller), so if you have the right WINZIP plugin for "tar.gz" files, it is advised to choose this one.

 

Installation

Note that if you are not using the CardTable platform, there are some prerequisites before being able to use OpenCard components. See the prerequisites page for more details.

For installing the Gemplus Generic CardServices component:

  1. first, download "gemplus-service-0.2.zip" or "gemplus-service.tar-0.2.gz" (see above),

  2. unpack the distribution in the directory containing OCF, this should provide new files and directories among which: "lib/gemplus-service-0.2.jar",

  3. insert "lib/gemplus-service-0.2.jar" into your CLASSPATH environment variable (JDK 1.x) or copy this JAR file into your "jre/lib/ext/" directory (JDK 1.2 or 1.3).

  4. develop your own CardService that leverage this library.

More information on how to use this component is available, see below.
